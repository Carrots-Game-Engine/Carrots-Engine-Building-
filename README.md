## 📖 Welcome to Carrots Engine
**Carrots Engine** is a free, open-source 2D and 3D game engine made to be fast, simple, and incredibly powerful.
We started with the rock-solid base of the **GDevelop Core**, and expanded it into a next-generation toolset. We've completely overhauled the workflow, adding professional-grade tools like Blueprint scripting, PBR materials, advanced animation systems, and Git integration to help indie developers and studios bring their biggest ideas to life.
## ✨ What's New? (Our Engine Features)
We have added a massive list of features to make game development faster, better, and more enjoyable:
### 🎨 Beautiful Graphics & Rendering
 * **Advanced Shading:** Full Shader System, cool Toon Shaders, and a robust Material System with PBR Materials.
 * **Environment:** Realistic Physical Lighting and a beautiful Water System.
 * **2D Enhancements:** Normal Map support to make your 2D sprites pop with 3D depth, plus Skew Support.
 * **Maximum FPS:** Built-in LOD (Level of Detail) System and FSR (FidelityFX Super Resolution) for buttery smooth performance.
 * **Core Upgrades:** Massive Graphics Upgrade and deep Rendering Improvements.
### 🏃‍♂️ Next-Level Animations & Rigging
 * **Smart Animation:** A brand-new Animation System powered by an Animation State Machine and seamless Animation Blending.
 * **Cinematics & Tracking:** A full Timeline System for cutscenes and a Path Tracker for complex movements.
 * **Rigging:** Built-in Bone Editor with IK (Inverse Kinematics) support for realistic character movement.
### 🛠️ Pro Editor Tools & Workflow
 * **Code Your Way:** Blueprint Graph for visual node-based logic, alongside full TypeScript Support.
 * **World Building:** Scene Folder System, Child Object System (Parenting), and Built-in 3D Objects (Cube, Plane, Sphere, Capsule, Cylinder).
 * **Smooth Workflow:** Live Reload During Preview to see changes instantly, Snap Transform Tools, and Local/World Transform Modes.
 * **Professional Management:** Built-in Git Version Control Integration, GitHub Workflow Integration, and a powerful Console Log System.
 * **Editor Polish:** Enhanced Editor Workflow, Improved Asset & Object Organization, and major Inspector Improvements.
### 💥 Audio, Effects & Extensions
 * **Visual Magic:** An Advanced Particle Editor for stunning VFX.
 * **Sound Design:** A dedicated Audio Object System with an Enhanced 3D Audio Workflow.
 * **More Power:** Added a variety of New Extensions to expand your game's capabilities.
### 🚀 Performance & Optimizations
 * **Next-Gen Speed:** MultiThreads support (Web Workers Beta) for heavy processing.
 * **Efficiency:** Deep Runtime Optimizations, Performance Improvements, and Rendering Optimizations.
